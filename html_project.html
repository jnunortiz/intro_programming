My Html Notes

A user gets information through a web browser that uses http protocol to query data to a server. A server is a computer with the sole purpose of storing and sharing information.

Inline Elements

<b>

This tag makes the content bold.

<em>

This tag makes the content be displayed in italics.

<br>

This tag
breaks the line.

<a>

This tag anchors a hyperlink to a text.

<img>

This tag places an image inline.

<span>

This tag is used to group inline elements.

Block Elements

<p>

This tag is used to create a paragraph.

<div>

This tag creates a block container.
